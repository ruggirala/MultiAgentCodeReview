Shared ReviewState

Pydantic v2 model threaded through every node. Every agent reads source_code, appends Findings, and (downstream) populates patch and tests. No string parsing between agents.

Conditional routing

The triage node sets needs_human_review = True when any Critical/High security finding exists. LangGraph routes through a human_review checkpoint node that surfaces the findings — today a logged warning the operator sees in the watcher; a stubbed extension point for a real interrupt.

* human_review fired on 25 / 31 pipeline runs in the load test — most PRs had Critical security findings.

What this measures (and what it doesn't)

  • Measures: recall on bugs we deliberately planted — the agent flagged at least one finding in every expected category.
  • Doesn't measure: precision. Each PR also touches real airflow code; surrounding findings (often valid) lack ground truth.
  • Granularity: "category caught" not "exact bug caught." Hand-spot-checks confirm 7 / 8 specific bugs were cited by name + CWE.

What we'd improve next

Three things we measured today, with the room we'd want to close.

Speed
93s today → < 30s

Today, the slowest review in our 30-PR test took just over a minute and a half. Most of that is the patch and tests agents waiting on GPT-4o. Run the analysis agents at the same time instead of one-after-another, and use a faster (cheaper) model for the lighter passes — that's the path to under 30 seconds.

Languages
1 Python → 3+

Today the agents only read Python. The parser we use already understands Java, JavaScript, and Go — what's missing is teaching the bug and style prompts about each language. Security findings are mostly the same across languages, so that agent can extend almost as-is.

Cost per PR
~$0.14 GPT-4o → < $0.05

Today, each PR costs about 14 cents in OpenAI tokens. Most of that goes to fully reviewing every file with the most expensive model. Route simple, low-risk diffs to the cheaper GPT-4o-mini and only reach for GPT-4o when something looks risky — that gets us under 5 cents.
